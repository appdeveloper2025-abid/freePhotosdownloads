## 1. UNSPLASH API (BEST - 50,000 requests/month FREE)

### Get Your Free API Key:
1. Go to: https://unsplash.com/developers
2. Click "Register as a developer"
3. Create account (free)
4. Click "New Application"
5. Accept terms
6. Fill form:
   - Application name: "4K Wallpapers"
   - Description: "Personal wallpaper website"
7. Click "Create Application"
8. Copy your "Access Key"

### Add to api-config.js:
```javascript
accessKey: 'YOUR_ACCESS_KEY_HERE'
```

**What you get:**
- 3 million+ professional photos
- 4K+ quality
- All categories (nature, city, animals, etc.)
